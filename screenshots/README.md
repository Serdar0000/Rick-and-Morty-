# Скриншоты приложения

## Как добавить скриншоты

1. Сделайте скриншоты приложения на эмуляторе/устройстве
2. Сохраните их в эту папку с названиями:
   - `characters_screen.png` - экран списка персонажей
   - `favorites_screen.png` - экран избранного
   - `favorites_sort.png` - экран избранного с меню сортировки (опционально)
   - `character_card.png` - крупный план карточки персонажа (опционально)

3. Скриншоты будут автоматически отображаться в основном README.md
